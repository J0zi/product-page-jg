# joziglobal.net

Marketing website for **JoziGlobal s.r.o.** — custom cloud platform development on AWS and Kubernetes.

Built with Next.js 16, React 19 and Tailwind CSS 4.

## Develop

```bash
npm install
npm run dev
```

The site runs at <http://localhost:3000>.

## Build

```bash
npm run build
npm start
```
